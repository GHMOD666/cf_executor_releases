# v7.0 更新内容
- 重大更新: 版本号跃升至 v7.0, 稳定版发布
- 输出管道: FIFO 改为 PTY 伪终端, 强制脚本行缓冲, 输出顺序与 raw shell 一致
- 持久化: 文件双保险机制, 关键设置同步到 /sdcard/cf_executor/.settings
- 修复: 大退重进后密钥和自定义路径丢失问题
- 修复: 切换脚本时旧卡密残留导致验证失败
- 修复: pipe 写入竞态条件
- 修复: 输出轮询间隔优化至 100ms + 多重强制滚动
- 安全: 魔改 UPX (CFX! magic), 公开 upx -d 无法脱壳
- 安全: 字符串 XOR 加密, 反调试/反注入/自校验
